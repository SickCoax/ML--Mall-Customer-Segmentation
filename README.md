# Customer Segmentation using Agglomerative Clustering

## 📌 Overview
- Performed customer segmentation using **Agglomerative Hierarchical Clustering**.
- Determined the optimal number of clusters using **Silhouette Score Analysis**.
- Applied **PCA** for cluster visualization.
- Identified **6 distinct customer segments** based on customer behavior.

## 🚀 Features
✔️ Data Preprocessing & Feature Scaling  
✔️ Hierarchical Clustering (Ward Linkage)  
✔️ Dendrogram Analysis  
✔️ Silhouette Score Evaluation  
✔️ PCA Visualization  
✔️ Customer Segment Discovery  

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-Learn
- Matplotlib

## 📊 Results
- Optimal Clusters: **6**
- Cluster Quality Evaluated using **Silhouette Score**
- PCA used for 2D visualization of customer segments

## 🎯 Business Applications
- Customer Segmentation
- Targeted Marketing Campaigns
- Customer Behavior Analysis
- Business Decision Making

## 📈 Conclusion
Agglomerative Clustering successfully grouped customers into meaningful segments, enabling better understanding of customer purchasing patterns and supporting data-driven business strategies.
